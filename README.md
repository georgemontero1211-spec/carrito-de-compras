# Carrito de Compras

## 📋 Descripción

**Carrito de Compras** es una aplicación web que permite al usuario agregar productos a un carrito de compras, ver el total acumulado y manejar los elementos que ha seleccionado. Facilita ver lo que se ha añadido, eliminar productos y tener una estimación del costo total antes de “comprar”.

## 🛠 Tecnologías usadas

- **React** → interfaz de usuario basada en componentes reutilizables y actualización reactiva.
- **TypeScript** → tipado estático para funciones, componentes y estados.
- **Vite** → servidor de desarrollo ágil con recarga en caliente y optimización de bundling.
- **PostCSS** → procesamiento de estilos CSS moderno.
- **ESLint** → reglas de estilo y detección temprana de errores de código.

## 📚 Conceptos aplicados

- **Componentes funcionales de React** → separar la UI en piezas independientes (producto, carrito, lista, etc.).
- **Estado local (useState)** → para gestionar lo que hay en el carrito y actualizar los totales conforme se agregan/eliminan productos.
- **Props entre componentes** → para pasar datos como producto, cantidad, precio, etc.
- **Tipado fuerte con TypeScript** → para mejorar seguridad del código y evitar bugs comunes.
- **Organización modular con tsconfig y estructura de carpetas** → facilita mantenimiento y escalado.
- **Linting con ESLint** → mantener estilo coherente y detectar posibles errores desde temprano.
